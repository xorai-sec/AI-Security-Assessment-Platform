# Roadmap

## Implemented in first release

- EnterpriseAssist laboratory target.
- Controlled native campaigns for prompt leakage, RAG injection, restricted retrieval,
  unauthorized tools, confirmation bypass, and cross-session memory leakage.
- Evidence capture, deterministic evaluation, risk scoring, ISO mapping candidates.
- Markdown, HTML, JSON, and CSV report generation.

## Planned

- PostgreSQL persistence with Alembic migrations.
- Redis-backed background workers.
- garak adapter execution and native report ingestion.
- PyRIT multi-turn objective campaigns.
- Promptfoo regression-suite export.
- Full React workflow pages and role-based authentication.
- PDF report rendering.
- vLLM high-throughput deployment profile.
