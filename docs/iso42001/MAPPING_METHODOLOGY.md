# ISO/IEC 42001 Mapping Methodology

No copyrighted ISO standard text is included.

The platform uses original placeholder mapping templates that connect:

```text
Technical observation
  -> AI risk
  -> Control objective
  -> Organizational process
  -> Required evidence
  -> Available evidence
  -> Evidence gap
  -> Potential conformity concern
```

Mapping outcomes are evidence candidates only. Human review is required before
audit conclusions are made.
