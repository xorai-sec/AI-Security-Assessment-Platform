# Client Presentation Script

## 20-30 minute flow

1. Explain the AI security and ISO/IEC 42001 assurance problem.
2. Open the Dashboard and show live API/framework state.
3. Open Targets and register EnterpriseAssist.
4. Register the OpenAI-compatible fixture.
5. Register the Custom REST fixture.
6. Validate and discover capabilities for each target.
7. Launch an authorized assessment against EnterpriseAssist.
8. Show real prompts, responses and telemetry in Evidence.
9. Show confirmed findings and ISO candidate mappings.
10. Launch an assessment against the OpenAI-compatible or Custom REST fixture.
11. Compare target results with `make assess-target-group`.
12. Open Markdown/HTML/JSON/CSV reports.
13. Explain safety controls, authorization requirements and current release limitations.

Use persisted assessment data from the current run. Do not present static screenshots as test results.
